CSS cheatsheet

Flexbox Row

Use these three properties to create a flexbox row layout

.row { display: flex; flex-direction: row; flex-wrap: wrap; }

Flexbox Column

Use this to creat a Flexbox column layout

.column { display: flex; flex-direction: column }

CSS Grid Layout

Build a 12 column layout using CSS Grid.

grid { display: grid; width: 100%; grid-template-columns: auto; repeat: (12, 1fr); }

LInear Gradients

This will creat a backgound linear gradient from top to bottom.

.linear-gradient-background { backround-image: linear-gradient{ rgba(0%) rgba(100%)}: }

Box Trasnsition Glow

Use transition and box shadows to glow on hover.

.code-card .card-header { border-radius: 8px; transition: all 0.5s ease-in-out;} .code-card:hover, . card-header { box-shadow: insert 0px 0px 8px rgba(0 0 0). 0 0 15px rgba(0 0 0):}

Overlay Card with Title

Use position properties and negative margins to raise elements higher than their natural starting point

.card-header { position: relative; margin-top: -20px; }
